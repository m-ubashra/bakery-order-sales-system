# Bakery Order and Sales Management System

This project is a Bakery Order and Sales Management System developed using Microsoft Access. It helps manage customer information, products, orders, and sales records through a simple and user-friendly database application.

## Features

- Customer Management
- Product Management
- Order Management
- Sales Records
- Reports
- User-Friendly Interface

## Technologies Used

- Microsoft Access
- SQL (MS Access Queries)
